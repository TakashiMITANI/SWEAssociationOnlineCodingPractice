# Docs Index

`docs/` は番号付きディレクトリで管理します。

## Naming Rule

- 形式: `{order}_{category}`
- 例: `00_setup`, `10_workflow`, `20_repository`
- `order` は10刻みで付与し、将来の追加余地を残す

## Current Categories
