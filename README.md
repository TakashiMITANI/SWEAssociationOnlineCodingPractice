# SWE Association Online Coding Practice

このリポジトリは、SWE Association のオンラインコーディング練習用リポジトリです。

## Purpose

- コーディング課題の作成・管理
- 解答例や検証コードの管理
- 学習用メモ・運用ルールの集約

## Structure

現時点では初期状態です。必要に応じて以下のような構成を追加してください。

- `problems/`: 課題文や入力例
- `solutions/`: 解答コード
- `tests/`: テストコード
- `docs/`: 補足資料

## Usage

1. 課題を `problems/` に追加する
2. 解答を `solutions/` に追加する
3. テストを `tests/` に追加し、実行して確認する
