# Codex Operation Guide

このリポジトリでは、Codex を主に Issue 管理と PR 作成支援に利用します。

## Allowed Tasks

- Issue の整理、テンプレート化、優先度付け
- Issue ラベル付与・更新（status / priority / type）
- PR 説明文の作成、チェック項目の確認
- コミットメッセージ整備とコミット実行（Issue関連付けルール準拠）
- ドキュメント更新（README / docs）
- リポジトリ運用ルールの整備

## Restricted Tasks

- LeetCode / arai60 の問題の解答に関する質問対応（全面禁止）
- 解法、ヒント、方針、擬似コード、計算量比較の提供
- 問題を「代わりに解く」行為

## Operation Policy

- Codex には「課題の管理・品質向上」を依頼する
- 課題の解答に関する依頼が来た場合は対応不可として扱う
- 代替として、Issue / PR / ドキュメント運用の依頼へ誘導する
- Issue ラベルは `docs/10_workflow/issue-labels.md` の単一選択ルールに従って更新する
- コマンド許可方針は `AGENTS.md` と `docs/00_setup/agent-command-permissions.md` に従う
